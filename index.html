My Projects

YouTube-Comments-Sentiment-Analysis

This project is designed to scrape YouTube comments from a specific video, analyze the sentiment of each comment using a machine learning model, and then save both the comments and their respective sentiment analysis results into CSV files. The project leverages the YouTube Data API and Hugging Face's transformers library to perform sentiment analysis..

Virtual-Whiteboard-By-Hand-Gestures

This project sets up a simple web application that utilizes MediaPipe's hands model to interact with a whiteboard. The application allows users to draw on a canvas using their hands as input, view webcam feed, and control drawing functions with a user interface.

ATM-Simulation-System

This is a simple simulation of an ATM machine implemented in Java. It allows users to view balance, withdraw, deposit, view a mini statement, and exit the ATM system.
